# About projects
Admin panel for managing a private clinic and tracking financial indicators.


## Stack used
- backend
  + Python
  + FastAPI
  + SQLAlchemy
  + Pytest
- frontend
    + Kotlin mutliplatform
    + Kotlin/JS 
    + Ktor client
    + Kvision (*web framework for Kotlin/JS*)


## Run project
0. Install docker and docker compose
1. Run:
```
docker compose up
```
